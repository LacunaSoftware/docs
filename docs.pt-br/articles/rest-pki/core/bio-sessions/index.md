# Sessões de biometria - Rest PKI Core

O **Rest PKI Core** oferece o serviço de sessões de biometria, uma solução para verificar a identidade de usuário por meio de caracteristicas físicas, como o rosto.

Uma sessão de biometria representa o ciclo completo de uma interação de verificação:

1. Início: O sistema solicita a verificação do usuário.
2. Captura: O usuário realiza a ação necessária (ex: olhar para a câmera seguindo as instruções na tela).
3. Conclusão: O sistema processa os dados e retorna um resultado que pode ser consumido pela sua aplicação.

Essa solução é indica para cenários que exigem maior segurança ou rastreabilidade na identificação de pessoas, como logins, abertura de conta, assinatura de documentos, autenticação para operações sensiveis.
